# 04. Hysteria2 — быстрый UDP-резерв

**Зачем:** TCP-маскировка (Reality) и UDP-протоколы ломаются ТСПУ **разными
способами**. Держа по одному представителю каждого типа, вы резко снижаете
вероятность полного «нуля». Hysteria2 — быстрый UDP/QUIC: отлично для игр и
видеозвонков, низкий пинг.

- **Salamander** — обфускация, ломает QUIC-детекцию ТСПУ (делает сервер
  несовместимым с обычным QUIC — это норма для обхода).
- **Port hopping** — клиент «прыгает» по диапазону портов; если режут пару
  IP+порт, следующее соединение идёт по другому порту.
- **Маскарад** — случайным гостям/сканерам сервер отвечает как прокси на донора.

## Развёртывание

```bash
sudo ./scripts/04-install-hysteria2.sh
```

Скрипт: ставит Hysteria2 (`get.hy2.sh`) → генерирует пароли obfs/auth → рендерит
[`configs/hysteria2/config.yaml.template`](../configs/hysteria2/config.yaml.template) →
настраивает port hopping через nftables → запускает → печатает hysteria2:// ссылку и QR.

## TLS: домен или самоподпись

- **Если задан `DOMAIN`** (A-запись на VPS) — выпускается реальный сертификат
  Let's Encrypt (ACME). Надёжнее, клиент без `insecure`.
- **Если домена нет** — генерируется самоподписанный сертификат; в клиентской
  ссылке добавляется `insecure=1`.

## Port hopping

`scripts/setup-port-hopping.sh` ставит nftables-правило: весь диапазон
`HY2_HOP_RANGE` (по умолчанию `20000:50000`) UDP перенаправляется на `HY2_PORT`.
Правило персистентно через systemd-сервис `vpn-hopping` (переживает перезагрузку).

> Минус: широкий открытый диапазон портов сам по себе может быть аномалией.
> Используйте осознанно. Чтобы выключить — оставьте `HY2_HOP_RANGE` пустым в `.env`.

В клиентской ссылке порт задаётся диапазоном: `…:443,20000-50000/…` — клиент сам
меняет порт каждые N секунд.

## Настройки под мобильные сети

- Если режут UDP вне 443 — используйте `HY2_PORT=443`.
- Полоса в конфиге (`bandwidth`) — выставляйте ~80% реальной полосы сервера.
- Пароль обфускации **обязан совпадать** на клиенте, иначе таймаут «как будто
  сервера нет».

## Файрвол

```bash
ufw allow 443/udp
ufw allow 20000:50000/udp     # для port hopping
```

`01-harden.sh` открывает их автоматически на основе `.env`.
