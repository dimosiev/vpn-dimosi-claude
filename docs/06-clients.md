# 06. Клиенты, подписки и автопереключение

Нужен **современный прокси-клиент** с поддержкой свежего Xray-core и flow Vision.
Обычные «классические» VPN-приложения не подойдут.

## Приложения (2026)

| Платформа | Рекомендуется |
|-----------|---------------|
| iOS | v2RayTun, Streisand, Hiddify |
| Android | v2RayTun, Hiddify, v2rayNG |
| Windows | v2rayN, Hiddify |
| macOS | v2RayTun, Streisand, Hiddify |

## Импорт одного профиля

**По QR (телефон):**
1. Установите приложение.
2. Запустите `sudo ./scripts/gen-client-link.sh` на сервере — он покажет QR.
3. В приложении: `+` → Scan QR → наведите на экран. Connect.

**По ссылке (ПК):**
1. Скопируйте `vless://…` из вывода скрипта.
2. В приложении: Import from clipboard / Добавить из буфера → Connect.

## Автопереключение между схемами (рекомендуется)

Чтобы не переключаться вручную, соберите в одном клиенте профили со **всех**
серверов и схем — клиент сам выберет живой по задержке (url-test). Это
автоматизация «обороны глубиной»: блокировка одной двери = автоматический переход
на другую.

**sing-box / Hiddify:**
```bash
sudo ./scripts/gen-singbox-client.sh     # → out/client-singbox.json
```
Импортируйте `out/client-singbox.json` в Hiddify (Add profile from file). В нём:
- профили **Reality + Hysteria2** одного сервера в группе `auto` (url-test, интервал 3 мин);
- трафик к `.ru`/`.рф` и приватным сетям идёт **напрямую**, мимо туннеля;
- ручной селектор для принудительного выбора.

Добавьте в `outbounds` профили второго сервера (просто скопируйте блок и
поменяйте `server`/ключи), впишите их теги в группы `auto` и `select` — получите
отказоустойчивость между странами.

## Подписки (subscription) — удобство при смене серверов

Вместо рассылки новых ссылок при каждой смене сервера дайте клиенту **одну
ссылку-подписку**. Меняете серверы на бэкенде — клиент обновляет подписку и
получает свежие конфиги автоматически. Это и есть «не остаться за бортом» в
автоматическом режиме.

Подписки умеют генерировать **3X-UI** и **Marzban** (см.
[08-defense-in-depth.md](08-defense-in-depth.md)). Для нескольких серверов и
пользователей Marzban предпочтительнее.

## Проверка после подключения

1. **IP сменился** — `https://ipinfo.io` показывает IP сервера.
2. **Нет утечек** — любой «VPN leak test»: нет утечек DNS / WebRTC / IPv6.
3. **Скорость адекватная** — прогоните спидтест.
4. **Маскировка работает** — `https://ВАШ_IP` напрямую (без клиента) показывает
   страницу донора или TLS-ошибку, но не что-то выдающее прокси.

Все 4 пункта ОК → у вас рабочий устойчивый VPN.
